# RULES — legal/

> Templates only. No signed documents. Requires explicit approval to write here.

---

## What belongs here

- Contract templates (unsigned).
- MOU and partnership agreement templates.
- Policy and compliance document templates.
- Legal reference materials (unsigned, non-sensitive).

## What does not belong here

- Signed contracts or agreements of any kind.
- Documents containing patient identifiers.
- Sensitive partner documents.
- Credentials, tokens, or access keys.

## Hard ban

> **No signed documents. No PHI. No exceptions.**

## Access rules

- Claude must **not** write to this folder without explicit user instruction.
- User must confirm the action before any file is created or modified here.
- Do not generate client-facing legal copy without approval.

## Naming convention

- Use `lowercase_with_underscores`.
- Add date prefix when relevant: `YYYY-MM-DD_name.ext`
- Mark templates clearly: `template_` prefix recommended.

## Before adding a file

- [ ] Confirmed this is a template, not a signed document.
- [ ] User has explicitly approved writing to `legal/`.
- [ ] No PHI or sensitive partner information included.
- [ ] Filename follows naming convention.
